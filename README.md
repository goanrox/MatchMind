# ⚽ MatchMind — AI Soccer Fan Education App

> Built by Xavier Serrao | World Cup 2026 Edition

## What It Does
MatchMind is an interactive soccer fan education platform 
built for the 2026 FIFA World Cup. It helps fans learn the 
game, test their knowledge, and get AI-powered insights — 
all in one place.

## Features
- 🏫 **Soccer School** — Learn the rules, positions, and 
  tactics from beginner to advanced
- 🎮 **Match Simulator** — Simulate match scenarios and 
  predict outcomes
- 🧠 **Fan IQ Quiz** — Test your soccer knowledge with 
  progressive difficulty
- 🗣️ **Jargon Translator** — Decode soccer terminology 
  in plain English
- 🤖 **Milo AI Chatbot** — Live AI-powered soccer assistant 
  (Groq API via Vercel serverless function)

## The Problem It Solves
With the World Cup coming to the US in 2026, millions of 
new fans want to understand the game but don't know where 
to start. MatchMind meets them where they are — fun, 
interactive, and powered by AI.

## Tech Stack
- Single-file HTML/CSS/JavaScript
- Groq AI API (via Vercel serverless function)
- Glassmorphism UI design
- Deployed on Vercel & GitHub Pages

## Architecture
Built as a self-contained single HTML file for maximum 
portability and performance — no framework dependencies, 
no build step required.
